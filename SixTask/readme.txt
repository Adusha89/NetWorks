2. HTTP и HTTPS
При передаче по протоколу HTTP данные не шифруются и передаются в
открытом виде. При передаче по HTTPS данные заворачиваются в TLS (см. скриншот)
и передаются в зашифрованном виде. Предварительно клиент-сервер устанавливают соединение
и обмениваются данными о ключах шифрования.
Сервисы google через браузер google используют протокол gquic (развернут поверх UDP)
Протокол использует возможности по скорости работы UDP и шифрования TLS
(передача данных идет на 443 порт (см. скриншот 3)).
